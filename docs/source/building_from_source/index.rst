Building cartopy from source
============================

The dependencies, with the minimum tested version, of cartopy |version| are:

 * cython - 0.15.1
 * proj.4 - version 
 * geos - 1.7.1
 * python2 - 2.7.3 
 * shapely - 1.2.14
 * pyshp - 1.1.4

    NOTE names of files should be in the form "cartopy-<version>__on__<os>-<os-version>.rst
